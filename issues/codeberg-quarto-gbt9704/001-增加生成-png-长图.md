# 增加生成 png 长图

## reply from @songwupei (2026-07-17)

已实现 `scripts/md2png.sh`，支持 Markdown / Quarto 文档 → PNG 长图。

**两种渲染管线：**
- **HTML 模式**（默认）— Quarto → HTML → Chrome 全页截图，emoji/CJK 渲染效果好
- **PDF 模式** — Quarto → PDF → pdftoppm → 合并长图

**功能亮点：**
- `--format` 支持扩展格式（如 `gbt9704-html` / `gbt9704-pdf`）
- 自动裁边 + 统一边距（`--margin`）
- 自动检测 Chrome / Chromium

```bash
# 基础用法
./scripts/md2png.sh document.md

# 使用 gbt9704-html 格式
./scripts/md2png.sh example.qmd --format gbt9704-html --width 1200
```

已在 v0.3.0 (3eb4eed) 中发布。
